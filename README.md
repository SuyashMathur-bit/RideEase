# RideEase

### Android-Based Cab Booking and Ride Management System

---

## Project Overview

RideEase is a full-stack Android cab booking application that enables users to book rides by selecting pickup and drop locations, view booking confirmations, and access their ride history.

The application integrates an Android frontend with a PHP/MySQL backend and uses Volley for API communication. It simulates the core functionality of modern ride-hailing platforms such as Uber and Ola, providing a seamless booking experience.

The primary objective of the project is to demonstrate mobile application development, location-based services, backend integration, and ride management workflows.
